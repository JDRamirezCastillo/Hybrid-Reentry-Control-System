# Hybrid-Reentry-Control-System
Code and simulation framework for 'Hybrid Aerothermodynamic Modeling and Control of Re-Entry Vehicles in LEO'.
## How to Run
1. Clone the repository:
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the simulation:
   `python reentry_sim.py`

## Results
The current configuration achieves an RMS gamma error of **0.1122 rad**, validated on consumer-grade hardware to ensure computational efficiency for flight-ready systems.
